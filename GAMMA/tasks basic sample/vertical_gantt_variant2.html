Vertical Gantt Chart Variant 2: True Vertical Time Axis (Date-Centric)

This variant is a more literal **vertical flip**, where the **time axis runs vertically** down the page. Each row represents a time unit (a day in this case). Tasks are represented as blocks that span multiple time units in their respective columns. This is still highly vertical-scroll friendly, but horizontal space is consumed by the number of parallel tasks (columns).
